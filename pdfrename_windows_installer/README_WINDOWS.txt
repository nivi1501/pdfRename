Prerequisite:

Java is the prerequisite for this application.


**********************************************
******************For Windows-10:*************
**********************************************
For single file:

--> Right click on the pdf --> select pdfrename_windows --> Choose proxy option yes or no --> If yes --> Enter Your proxy details --> Renaming process starts.
											|
											|---> If no ---> Renaming process starts.


For multiple files:

--> Right click on the pdf --> Select "sendto" in right click menu --> Choose proxy option yes or no --> If yes --> Enter proxy details --> Renaming process starts.
														|
														|---> If no --> Renaming process starts for multiple files.

*****************************************
*************For Windows-11:************
***************************************

For single files:

--> Right click on the pdf --> Select more options --> Select Pdfrename_windows --> Choose proxy option Yes or no --> If yes --> Enter your proxy details --> Renaming process starts.
																|
																|---> If no --> Renaming process starts.

For multiple files:

--> Right click on the pdf --> Select more options -->select "sendto" in right click menu -->Choose proxy option yes or no --> If no --> Renaming process starts for multiple files.
																		|
																		|---> If yes --> Enter proxy details --> Renaming process starts.

Note: Suppose we would like to rename the 'xyz.pdf' file. The file should be renamed to 'au_2012.pdf' after processing. Suppose that there is already a file named "au_2012.pdf" 
	in my folder. In this case, 'xyz.pdf' cannot be renamed because we already have a file with the correct name.
